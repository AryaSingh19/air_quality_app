# Air Quality Project

A Python-based application to check and visualize air quality data using Streamlit.

---

## Features

- Fetches air quality data from APIs or local sources
- Displays AQI (Air Quality Index) with color-coded indicators
- Visualizes pollutant levels using charts
- Easy-to-use interactive interface via Streamlit

---

## Requirements

- Python 3.8 or higher
- Streamlit
- Pandas
- Matplotlib
- Requests (if fetching API data)
- Other dependencies listed in `requirements.txt`
